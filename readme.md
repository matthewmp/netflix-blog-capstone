## Background

My family and friends are always talking about what we are binge watching and giving each other recomendations.  I thought it would be easier to keep each other up to date with a forum dedicated to our love of Netflix.

## Summary

Netflix Junkie is a forum where a user can search for existing threads, create new topics & threads, can create/edit posts, like/unlike existing posts, and make comments on other users posts.
